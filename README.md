# JasHRadios

A feature-rich web application for streaming online radio stations, built with React, Vite, Tailwind CSS, and hls.js.

## Features

-   **Live Radio Streaming**: Supports standard audio and HLS (.m3u8) streams.
-   **Station Management**: Browse, search, and filter a curated list of Tamil radio stations.
-   **Favorites**: Mark stations as favorites for quick access (persisted locally).
-   **Responsive Design**: Modern, glassmorphism-inspired UI that works on all devices.
-   **Audio Visualizer**: Animated indicators for playback state.

## Run Locally

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is optimized for deployment on **Vercel**.

1.  Build the project:
    ```bash
    npm run build
    ```

2.  The output is located in the `dist` directory.
